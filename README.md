<h1> Projemdeki amaç sesli copilot uygulaması yapmaktır. End tuşuna bastığınızda sesinizi dinlemeye başlar ve tekrar end tuşuna basında sesinizi yazıya çevirip belirttiğiniz koordinattakı yazı yazma alanına girer. Eğer ses olarak sadece "Kabul" derseniz Allow butonuna tıklamış olursunuz. Editör copilotlarını sağ tarafa sabit koyunuz ve yazı yazma ve gönderme buton koordinatlarını programa giriniz.</h1>
